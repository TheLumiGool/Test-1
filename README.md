# Test-1

Roblox badminton prototype with modular OOP gameplay scripts.

## Setup
1. In Roblox Studio, create matching folders in **ReplicatedStorage**, **ServerScriptService**, and **StarterPlayer/StarterPlayerScripts**.
2. Copy the Lua scripts from `src/` into their respective services:
   - `src/ReplicatedStorage/Badminton` → `ReplicatedStorage/Badminton`
   - `src/ServerScriptService/Badminton` → `ServerScriptService/Badminton`
   - `src/StarterPlayer/StarterPlayerScripts/Badminton` → `StarterPlayer/StarterPlayerScripts/Badminton`
3. Play the game in Studio.

## Controls
- **E**: Serve
- **F**: Swing
